Bob's Adoption Center

A faux adoption site for furr babies. This project uses React within the Next.js framework, as well as CSS, SQLite, Bycrypt, express.js, jest.js and Playwright testing. This project was collaborated on with a team using agile methodologies and has a Continuous Integration and Deployment Pipeline implemented. We've deployed on Heroku and Vercel, containerizing our app with Docker, and have incorporated our tests upon each pull request using SonarCube and GitHub Actions.

World Cultures API

An API that focuses on the culture of countries around the world, with a hand curated music playlist on the /musics route. Created using JavaScript, Express.js, Sequelize, SQLite3, Bcrypt, OAuth, and tested and documented with Postman.

Warehouse App

A fullstack app for warehouse managers and truck drivers to see what products are shipping from and to which warehouse and manage which boxes and palettes are loaded into the warehouse by limiting by max capacity. The app has full CRUD so users are able to add, delete, update, and create palettes and boxes for each warehouse. This app uses JavaScript, HTML, CSS, Node.js, express.js, sequeslize, SQLite, Jest.js, and Handlebars.
